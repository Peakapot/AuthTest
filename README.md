# AuthTest - Simple Signup/Login Website

This project is a small Flask website that includes:

- A styled landing page
- User signup
- User login/logout
- Personalized welcome message after login
- A members-only page protected by authentication

## Run locally

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Then open http://127.0.0.1:5000 in your browser.
